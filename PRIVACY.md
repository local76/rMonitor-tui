rMonitor Privacy Policy

rMonitor (Rust TUI System Monitor) is built with a commitment to user privacy. As a local terminal application, rMonitor does not track, collect, or transmit your personal data.

Data Collection and Telemetry
rMonitor does not include any analytics, crash reporting platforms, or tracking metrics.
rMonitor does not collect or request any personal identifiers, email addresses, or system credentials.

Network Connections
rMonitor runs completely offline and does not make any network connections. The application only queries the local Windows host to retrieve system diagnostics, processes, and service states.

Local Data Storage
All configurations managed by rMonitor are kept strictly on your local machine under your user profile directory:
Configurations: Stored at %APPDATA%\rMon\config.yaml
Log Files: Diagnostics are logged locally to %APPDATA%\rMon\log.txt

rMonitor does not upload your local configuration files or logs to any remote server.
